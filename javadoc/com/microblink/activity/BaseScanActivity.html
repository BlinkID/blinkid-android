      • EXTRAS_OPTIMIZE_CAMERA_FOR_NEAR_SCANNING

        public static final java.lang.String EXTRAS_OPTIMIZE_CAMERA_FOR_NEAR_SCANNING
        Flag which indicates PhotoPay should optimize camera parameters for near object scanning. When camera parameters are optimized for near object scanning, macro focus mode will be preferred over autofocus mode. Thus, camera will have easier time focusing on to near objects, but might have harder time focusing on far objects. If you expect that most of your scans will be performed by holding the device very near the object, turn on that parameter. By default, this parameter is set to false, except in cases when from other settings it can be safely concluded that it is better to turn it on.
        See Also:
        Constant Field Values

      • EXTRAS_VIEWFINDER_MOVEABLE

        public static final java.lang.String EXTRAS_VIEWFINDER_MOVEABLE
        Flag that indicates whether viewfinder frame on screen moves when photopay detects payslip or not. If boolean true is written under this key, viewfinder is moveable. If it's false, viewfinder is fixed. If nothing is written, default behaviour is used, which depends on the country.
